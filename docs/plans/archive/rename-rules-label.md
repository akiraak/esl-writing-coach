# 「AIルール」の表記を「ルール」に短縮

## 目的・背景

グループ見出しを「AI」に短縮したため、その配下のボタン「AIルール」は AI が重複する。
ボタンとダイアログ見出しの表記を「ルール」にする。

## 対応方針

- `index.html`
  - ボタンラベル「✎ AIルール」→「✎ ルール」、title 属性「AIルールを編集」→「ルールを編集」
  - ダイアログ見出し「AIルール」→「ルール」

## 影響範囲

- index.html の表示テキストのみ（CSS クラス名・ID は変更しない）

## テスト方針

- ヘッドレス Chromium でボタン・ダイアログ見出しの表記とスクリーンショットを確認
