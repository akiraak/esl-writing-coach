# AIルールの説明文に「空欄でも大丈夫」を先頭で明記

## 目的・背景

AIルールダイアログの説明文はルールの書き方だけを述べており、必須入力に見える。
ルールが空でも添削は動くので、説明文の最初に「空欄のままでも大丈夫」であることを明記して入力の心理的ハードルを下げる。

## 対応方針

- `index.html` の `.rules-help` の文面を変更（先頭に空欄可の一文を追加）
  - 「空欄のままでも大丈夫です。ルールを書くと、AI がそれに沿って添削とアドバイスを行います。文体・レベル・語数などを自由に指定できます。」

## 影響範囲

- index.html の説明文 1 行のみ

## テスト方針

- ヘッドレス Chromium でダイアログを開き、説明文に「空欄のままでも大丈夫」が含まれることとスクリーンショットを確認
