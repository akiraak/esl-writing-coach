# 入力欄と添削英文に word 数を表示

## 目的・背景

英作文の練習では「〜語以上 / 以内で書く」といった分量の目安が重要。現在は語数を知る手段がないため、
**ユーザーの英作文（入力欄）** と **AI の添削済み英文（出力欄）** のそれぞれに word 数を常時表示する。

## 対応方針

- 各ペインのヘッダー（`ユーザーの英作文` / `AI の添削済み英文`）の右端に `123 words` 形式で表示する
- word のカウント方法: テキストを空白で分割した非空トークン数（`text.trim().split(/\s+/).filter(Boolean).length`）
  - 日本語混じりの下書きでは日本語の塊も 1 トークンと数えるが、目安表示なので許容する
- 更新タイミング:
  - 入力欄: `input` イベントごとに即時更新（自動保存のデバウンスとは独立）
  - 添削英文: 記事ロード時と添削結果の反映時に更新

## 影響範囲

- `backend/public/article.html`: 2 つのペインヘッダーに word 数表示用の `<span>` を追加
- `backend/public/css/style.css`: ヘッダー内の右寄せ配置と表示スタイルを追加
- `backend/public/js/article.js`: word 数計算・表示更新関数を追加し、ロード時 / 入力時 / 添削反映時に呼び出す
- バックエンド・API: 変更なし

## テスト方針

- `./run-server.sh` で起動し、ブラウザ（ヘッドレス Chromium のスクリーンショット）で確認
  - 既存記事を開くと入力欄・添削英文の両方に word 数が表示されること
  - 入力欄への入力で即時に数が変わること（DevTools/スクリプトから input イベントで確認）
- `npm run typecheck`（バックエンドに影響がないことの確認）
