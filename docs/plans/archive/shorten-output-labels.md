# 出力ペインの見出しを短縮（AI の添削済み英文 → 添削済み英文 / AI からのアドバイス → アドバイス）

## 目的・背景

出力側のカラムには既にグループ見出し「AI のアウトプット」が付いているため、各ペインの見出しに
「AI の / AI からの」を重ねて付けるのは冗長。ペイン見出しを短くして読みやすくする。

## 対応方針

- `backend/public/article.html` のペイン見出し 2 箇所を変更する
  - `AI の添削済み英文` → `添削済み英文`
  - `AI からのアドバイス` → `アドバイス`
- 表示ラベルのみの変更。ID・クラス・JS・CSS・API は変更しない（該当文字列の出現は article.html の 2 箇所のみと grep で確認済み）

## 影響範囲

- `backend/public/article.html` のみ

## テスト方針

- ブラウザ（ヘッドレス Chromium のスクリーンショット）で見出しが変わっていること、word 数表示などが壊れていないことを確認
